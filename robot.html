The Omnidirectional Drivetrain

We utilize a 4-wheel mecanum drive system, powered by four NeveRest 40 motors. This design allows for omnidirectional movement, enabling precise lateral positioning and faster field navigation crucial for competitive matches. The chassis is constructed from custom-cut aluminum extrusion for maximum rigidity.

  • Motors: 4x NeveRest 40
  • Wheels: 100mm Mecanum
  • Speed: 1.5 meters per second

High-Reach Linear Slide

Our primary scoring mechanism is a 3-stage continuous linear slide actuated by a high-torque DC motor. It's designed to reach the top-most position of the scoring area (the 'Backdrop') with minimal deflection. The unique gripper tip ensures secure pixel release.

  • Actuation: 1x 5202 GoBILDA Motor
  • Extension: 30 inches of vertical travel
  • Precision: PID control for accurate height stopping

Control System & Software

The robot runs on a Rev Control Hub and is programmed entirely in Java using the FTC SDK. We heavily implement custom classes for drive control and state machines for reliable autonomous routines. Autonomous navigation relies on both AprilTags and Tensor Flow object detection.

  • Hardware: Rev Control Hub, 2x Servos, 4x Motors
  • Programming: Fixed Auto
  • Source Code: Available on GitHub
